Socratic questioning

การตั้งคำถามกับโลก การตั้งคำถามกับความเชื่อของตัวเองอยู่เสมอ กระหายที่จะหาคำตอบที่ดีกว่าด้วยหลักเหตุผลนิยม การช่วยกันหาคำตอบเป็นกลุ่ม โต้เถียงเพื่อคำตอบที่ดีที่สุด มิใช่เพื่อชัยชนะของตัวเอง ฯลฯ .. โสเครติสเชื่อว่าทั้งหมดนี้เป็นทักษะพื้นฐานของมนุษย์ที่มีปัญญาทางความคิด ผมก็เชื่อเช่นกันว่าปัญญาของมนุษย์ เกิดมาจากพื้นฐานความคิดที่ดี

Life, Liberty and Possessions.

มนุษย์ทุกคนเท่าเทียมและมีอิสระ ไม่มีใครควรจะมาทำลายชีวิต เสรีภาพ และทรัพย์สิน ซึ่งเป็นสิทธิธรรมชาติที่มนุษยมีมาตั้งแต่เริ่มเผ่าพันธุ์ ผู้ปกครองถูกตั้งขึ้นมาจากผู้ใต้ปกครอง เพื่อให้รักษาสิทธิทางธรรมชาติของพวกเค้า มิใช่ริดรอนมันไปจากพวกเค้า ฯลฯ .. ทั้งหมดนี้คือแนวคิดของ John Locke ที่ผมชอบมาก และคิดว่าเป็นสิ่งที่ไม่ใช่แค่นิสิต แต่เป็นคนไทยทุกคนที่ควรรู้จักไว้


ประเทศไทยมีปัญหาอย่างมาก ในด้านพื้นฐานความคิด และความเขื่อในสังคม เมื่อพื้นฐานความคิดถูกแก้ไข คนไทยคิดและเชื่อด้วยเหตุผล จนเกิดเป็นสังคมแห่งสติและปัญญา ผมเชื่อว่าผลลัพธ์ในด้าน creativity, innovation etc. จะเกิดตามมาเอง เหมือนกับที่ยุโรปสมัยภูมิธรรม หรือยุคก่อนปฏิวัติอุตสาหกรรมเคยเป็น



ต้องขออภัยด้วยจริงๆครับ พอดีผม copy คำถามมาไว้ใน Notes แล้วพิมพ์คำตอบ และไม่คิดว่ามันจะมี limit ใน textarea อยู่ด้วย ผมเลยตอบมาเยอะพอสมควร รบกวนลองอ่านเวอร์ชั่นเต็มดูได้ครับ


ข้อ 8

ผมคิดว่า ก่อนอื่นต้องตั้งคำถามสำคัญกับ jack ก่อนว่าพันธกิจของ alibaba คืออะไร. เป้าหมายที่ alibaba และ jack มุ่งหวังจากการเปิดตัวในเมืองไทยมีอะไรบ้าง. และแผนงานการเปิดตัวในประเทศไทยที่ HQ ได้วางแผนไว้แล้วนั้น รายละเอียดงานทั้งหมดเป็นอย่างไร. และแผนงานส่วน Mobile Application มีอะไรที่ต้องดำเนินการบ้าง. หรือถ้า HQ ยังไม่มีแผนงานการเปิดตัวในประเทษไทย alibaba team ในประเทศไทยก็ต้องมาคุยกันในช่วงนี้ ว่าสรุปแล้ว พันธกิจ และเป้าหมายที่ alibaba กับ jack ต้องการจากการเปิดตัวในประเทศไทยมีอะไรบ้าง แล้วผลสรุปที่ได้มานั้น เราจะมีแผนงานการเปิดตัวในประเทศไทยอย่างไรกันดี เพื่อให้บรรลุวัตถุประสงค์ขององค์กร(และตัว jack ด้วย) พอได้ big plan แล้ว ก็มาสับแผนงานสำหรับส่วน Mobile Application ว่าจะต้องมีแผนงานอะไรบ้าง ที่ผมและทีมต้องรับผิดชอบกัน และต้องรับผิดชอบให้เสร็จภายในเมื่อไร

ซึ่งถ้าจะคิดตามของจริงๆแล้ว ผมไม่รู้ว่าการทำงานจริงๆกับ alibaba จะเกิดอะไรขึ้นบ้างผมไม่มีประสบการณ์เลยครับ แต่ผมคาดว่าแค่คุยกันแค่นี้ เวลา 7 วันก็น่าจะหมดไปเรียบร้อยแล้ว๕๕๕ ซึ่งมันยังไม่ค่อยเกี่ยวข้องกับ mobile app เท่าไร ดังนั้นผมจะขอ assume ว่าระยะเวลาของงานที่ผมจะกล่าวต่อไปนี้ เป็นระยะหลังจาก big plan ของ top-level manager เสร็จแล้ว (ซึ่งช่วงนั้นผมก็คงไปประชุมกับพวกนี้ ไม่ได้ทำอะไรอย่างอื่น) และแต่ละ task จะเสร็จภายในเวลาที่ผมคาดคะเนไว้

day 1 : อธิบาย big plan ให้ team ฟัง, กำหนดเป้าหมาย และแผนงานระยะสั้นในช่วงสัปดาห์แรก ว่าต้องทำอะไรกันบ้าง
day 2-4 : วิเคราะห์สภาพแวดล้อมของตลาด e-commerce ในประเทศไทย, ศึกษา e-commerce application ของคู่แข่งในตลาด หาจุดเด่นและจุดด้อยของ app คู่แข่ง
รวบรวมข้อมูลพฤติกรรมการใช้งาน e-commerce application ของคนไทย เพื่อนำมาเป็นข้อมูลส่วนหนึ่งสำหรับการดีไซน์ UI, UX
+ ศึกษา requirement ของ application จาก HQ ควบคู่กันไป
day 5-6 : สรุปข้อมูลและ requirement, วิเคราะห์ข้อมูลและความเป็นไปได้, วางแผนงานของการ analysis, design และ implementation
day 7 : สรุปแผนและกำหนด deadline ของงานแต่ละงาน, เริ่ม execute แผน
